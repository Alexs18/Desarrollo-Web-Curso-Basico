Pasos para la Creación de un Nodo

  • Se crea la etiqueta con las propiedades createElement() y se lo almacena en una variable

    • Ejemplo: var NuevoNodo = document.CreateElement("p");

  • Se le asigna un contenido a la etiqueta por medio de la propiedad CreateTextNode() y le colocamos el contenido

    • Ejemplo: var Contenido = document.CreateTextNode("Este es un nuevo msj");

  • Luego se le asigna el contenido a el nodo creado con anterioridad

    • Ejemplo: NuevoNodo.appendChild(Contenido);

Pasos para la eliminación de un nodo

  • Procedemos a Localizar el nodo que queremos eliminar
  • Lo guardamos en una variable

    • ejemplo var NuevoEliminado = document.getElementById("NuevoNodo");

  • Procedemos a removerlo

    • NuevoEliminado.parentNode.removeChild(NuevoEliminado)

    Cabe destacar que los eliminados pueden ser accedidos por todas las propiedades que se visualizaron en Esquemas
