Insercion

Para Realizar la inserción de un nuevo nodo en la estructura arbol debemos seguir los siguientes pasos

  • Definir el Lugar donde se quiere insertar el nodo
  • llamar el elemento, mediante id, clase o nombre de donde se quiera ingresar el elemento
  • Insertar el Nuevo Nodo en base a las instrucciones establecidas en el apartado Pasos
